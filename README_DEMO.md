# 家庭车库智能充电机器人 - 路径跟踪控制仿真

## 项目简介

本项目实现了一个**全向轮移动机器人**在车库环境中的**路径跟踪控制仿真系统**，适用于：
- 🏆 大学生机器人竞赛
- 📚 控制算法课程设计
- 🔬 移动机器人导航研究

### 核心特性

✅ **模块化设计** - 清晰的代码结构，易于理解和扩展  
✅ **性能评估** - 10+ 项控制性能指标  
✅ **参数对比** - 自动化参数优化实验  
✅ **状态机** - 完整的充电任务流程  
✅ **高质量可视化** - 适合PPT展示的图表

## 技术方案

### 机器人平台

- **底盘类型**：三轮全向轮（正三角形布局）
- **车轮配置**：0°, 120°, 240°
- **运动学模型**：全向运动学逆解

### 控制算法

- **控制器类型**：位置-航向解耦控制
- **位置控制**：比例控制（可调增益）
- **航向控制**：比例控制（可调增益）
- **前视距离**：可配置（影响轨迹平滑度）

### 性能指标

| 指标类别 | 指标名称 | 单位 | 说明 |
|---------|---------|------|------|
| 位置跟踪 | 平均位置误差 | m | 整体跟踪精度 |
| 位置跟踪 | 最大位置误差 | m | 最坏情况性能 |
| 位置跟踪 | 最终到位误差 | m | 定位精度 |
| 航向跟踪 | 平均航向误差 | ° | 姿态控制精度 |
| 路径效率 | 路径效率 | % | 路径跟随效率 |
| 时间性能 | 总仿真时间 | s | 任务完成时间 |

## 参数调优指南

### 位置控制增益 (position_gain)

- **作用**：控制对位置误差的响应速度
- **推荐范围**：1.0 - 2.5
- **调优建议**：
  - 增大 → 快速纠正偏差，但可能振荡
  - 减小 → 更平滑，但响应慢

### 航向控制增益 (heading_gain)

- **作用**：控制对航向误差的响应速度
- **推荐范围**：1.5 - 3.0
- **调优建议**：
  - 增大 → 快速对齐路径方向
  - 减小 → 转向更平滑

### 前视距离 (look_ahead_distance)

- **作用**：控制跟踪点的前瞻距离
- **推荐范围**：0.5 - 1.2 m
- **调优建议**：
  - 增大 → 轨迹更平滑，但可能切弯
  - 减小 → 更紧密跟随路径

## 常见问题

### Q1: 如何修改路径类型？

编辑 `src/config.py`：
```python
path_type: str = "L_shape"  # 可选: "straight", "circle", "L_shape"
```

### Q2: 如何调整仿真时间？

编辑 `src/config.py`：
```python
total_time: float = 30.0  # 仿真总时长（秒）
dt: float = 0.05          # 仿真步长（秒）
```

### Q3: 如何获得更高分辨率的图表？

在 `run_demo.py` 中修改 DPI：
```python
dpi=300  # 默认300，可改为600获得更高分辨率
```

### Q4: 如何添加自定义路径？

在 `src/path_planner.py` 中添加新方法：
```python
@staticmethod
def _generate_custom_path() -> np.ndarray:
    # 自定义路径生成逻辑
    x = ...
    y = ...
    return np.column_stack([x, y])
```
